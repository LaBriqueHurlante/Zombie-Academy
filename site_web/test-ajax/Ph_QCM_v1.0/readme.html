Ph_QCM 1.0 : Readme

Introduction

Ph_QCM est un script écrit en PHP5 qui permet de créer facilement des QCM sur internet grace à une interface conviviale. Les QCM peuvent ensuite être consultés et remplis par les internautes. A la fin du QCM, les résultats sont donnés, ainsi qu'une correction pour chaque question. Ph_QCM n'utilise pas de base de données, ce qui permet de l'installer très facilement et de pouvoir sauvegarder le contenu des QCM à tout instant : il suffit de copier le dossier du script.

Installation

Pour installer Ph_QCM, éditez le fichier "_config.php" présent dans le dossier Ph_QCM et modidiez le mot de passe à votre guise. Copiez ensuite simplement le dossier "Ph_QCM" sur un serveur supportant PHP5 et le tour est joué.

Pour plus de sécurité, si vous utilisez Apache comme serveur, vous pouvez copier le fichier ".htaccess" présent dans le dossier "protection" dans le dossier du script ("Ph_QCM" par défaut). Les fichiers XML de vos QCM ne seront plus accessibles depuis un navigateur. Cela peut éviter que certains utilisateurs ne lisent les réponses avant de répondre au QCM.

Remarque pour free : pour installer le script chez free, vous devez créer un dossier "sessions" à la racine de votre site afin que les sessions PHP fonctionnent correctement. De plus free utilise encore (11/08/2006) PHP4 par défaut pour exécuter les fichiers *.php (et non PHP5, qui est utilisé par ce script). Pour remédier au problème copier le fichier ".htaccess" présent dans le dossier "free" dans le dossier du script ("Ph_QCM" par défaut). Si vous avez déjà copié le premier fichier ".htaccess" du dossier protection, ajoutez le contenu du deuxième fichier au premier.

Utilisation

Pour utiliser le script, rendez-vous sur la page principale du script. Là vous pouvez ouvrir une session Admin en entrant votre mot de passe dans la zone de texte précédée par "Admin >>". Vous pouvez ensuite ajouter un nouveau QCM en cliquant sur "[Créer un nouveau QCM]", puis modifer ce dernier en cliquant sur "Editer le titre" puis "Editer les questions" (Boutons situés sous le cadre du QCM). Une fois le QCM terminé, vous pouvez le visualiser en cliquant sur le titre.
